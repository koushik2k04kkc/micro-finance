# Micro Finance

A lightweight micro-finance project for managing loans, repayments, and customer accounts.

## Features

- Track customers and accounts
- Manage loan disbursements and repayments
- Record transaction history
- Generate simple summaries and reports

## Getting Started

1. Clone the repository.
2. Install dependencies.
3. Run the application.

## Usage

Use the provided scripts or commands in the project to launch the service and interact with the finance data.
